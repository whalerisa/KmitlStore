How to edit profile(วิธีการแก้ไขโปรไฟล์) สามารถทำตามขั้นตอนง่าย ๆ ได้ดังนี้:

วิธีการแก้ไขโปรไฟล์ใน KMITL Store:

  1. กดเข้าไปที่หน้า profile
  2. เลือกไปที่ปุ่ม icon Edit รูปดินสอตรงข้าง username
  3. เมื่อกดไปที่ปุ่ม icon Edit รูปดินสอแล้วจะไปขึ้นที่หน้า edit Profile เพื่อทำการแก้ไข profile
  4. ข้อมูล profile ที่สามารถแก้ไขได้จะมี ดังนี้
    • ข้อมูล username
    • ข้อมูล phone number
    • ข้อมูล email

    หมายเหตุ:

  5. หากทำตามขั้นตอนขั้นต้นที่กล่าวไปแล้วไม่สามารถเป็นไปตามที่ต้องการสามารถติดต่อทางด้าน admin เพื่อแก้ปัญหาโดยตรงได้ทางช่องทางนี้
