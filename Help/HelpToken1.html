How to register Login (การลงทะเบียนการเข้าสู่ระบบ) สามารถทำตามขั้นตอนง่าย ๆ ได้ดังนี้:

การลงทะเบียนการเข้าสู่ระบบ KMITL Store:

  1. เมื่อต้องการเข้าสู่ระบบสามรถ loginได้โดยการกดไปที่หน้า profile หากยังไม่ได้ login ระบบจะให้ไปที่หน้า login
  2. หากมีบัญชีอยู่แล้วสามารถ login ได้เลยแต่ถ้ายังไม่มีการสร้างบัญชีมากก่อนจะต้องไปที่หน้า register เพื่อสร้างบัญชีก่อน จากนั้นจึงสามารถ loginได้
  3. โดยสรุปเป็นขั้นตอนง่ายๆดังนี้
    • เข้ามาที่หน้า profile
    • หากยังไม่มีการ login บัญชีผู้ใช้ไว้จะมีข้อความให้กดไปที่หน้า login เพื่อเข้าใช้งาน
    • เมื่อถึงหน้า login หากเคยสร้างบัญชีไว้แล้วสามารถกรอก username และ password เข้าใช้งานได้เลย
    • หากยังไม่เคยมีการสร้างบัญชีจะต้องกดไปที่หน้า register และกรอกข้อมูลเพื่อสร้างบัญชีเข้าใช้งานเว็บไซต์
    • เมื่อกรอกข้อมูลหน้า register เพื่อสร้างบัญชีผู้ใช้เรียบร้อยแล้วก็สามารถไปที่หน้า login เพื่อ login เข้าใช้งานได้เลย
